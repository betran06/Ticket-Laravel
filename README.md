# 🎫 Ticketing System Backend API

Ticketing System Backend API is a web service built with Laravel 12 designed to manage support tickets efficiently within an organization.

The system allows users to create, track, update, and resolve support tickets while maintaining structured workflows and role-based access control.

---

# 🚀 Features

### 🎫 Ticket Management
- Create support tickets
- Update ticket status
- Assign tickets to staff
- Track ticket progress
- Manage ticket priorities
